Проект на [Next.js](https://nextjs.org) (React), [Tailwind](https://tailwindcss.com). Деплой через [Vercel](https://vercel.com)

## Первоначальная настройка

Установка зависимостей
```bash
bun install
```

Запуск дев сервера

```bash
bun dev
```

Сервер будет запущен [http://localhost:3000](http://localhost:3000)
